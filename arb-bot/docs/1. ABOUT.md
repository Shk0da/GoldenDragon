Этот код представляет собой **смарт-контракт на Solidity для арбитража с использованием флэш-кредитов Aave V3**.

## Основная функция контракта

Контракт совершает **двухэтапный арбитраж**:
1. Берет флэш-кредит в Aave (один актив)
2. Обменивает его через два разных роутера Uniswap V2
3. Возвращает кредит + комиссию Aave, забирая прибыль

## Ключевые компоненты

**ArbPlan** - структура с торговой стратегией:
- Два роутера для обменов
- Два пути обмена (токен → промежуточный токен → исходный токен)
- Минимальный выход и минимальная прибыль

**Процесс работы:**
1. `startArbitrage()` - владелец запускает арбитраж с планом
2. `executeOperation()` - Aave вызывает эту функцию после выдачи кредита
3. Внутри колбэка выполняются два обмена через разрешенные роутеры
4. Проверяется, что прибыль ≥ запланированной
5. Кредит возвращается Aave

## Механизмы безопасности

- **Белые списки** разрешенных токенов и роутеров
- **Механизм паузы** - можно остановить все операции
- Проверка целостности плана (`_checkPlan`)
- Защита от повторного входа (`loanOpen`)
- Восстановление токенов владельцем при паузе

## Что именно делает код

Это автоматизированный бот для **межбиржевого арбитража**: находит разницу цен между двумя DEX, берет беспроцентный кредит (если вернуть в рамках одной транзакции), покупает дешевле на одной бирже, продает дороже на другой, возвращает кредит с комиссией (~0.09%) и забирает разницу.

**Важно:** Это рабочий, но непроверенный код - содержит рискованные допущения (например, отсутствие проверок slippage в одном месте). Не рекомендуется использовать без аудита.
